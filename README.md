## 📖 Sobre o Projeto
Este projeto nasceu da vontade de unir tecnologia e sentimento. É um site desenvolvido do zero para a **Maria Vitória**, focado em proporcionar uma experiência visual agradável, com animações suaves e total responsividade (funciona perfeitamente no celular e no computador).

## 🚀 Tecnologias e Conceitos Aplicados
Para este projeto, utilizei o trio fundamental do desenvolvimento Web, aplicando boas práticas de UI/UX:

- **HTML5**: Semântica estrutural para acessibilidade.
- **CSS3**: 
  - Flexbox e Grid para layout.
  - Animações customizadas (`@keyframes`).
  - Media Queries para design responsivo.
- **JavaScript (Vanilla)**: 
  - Manipulação de DOM para interatividade.
  - Lógica de eventos dinâmicos.

## 📱 Funcionalidades
- [x] Design totalmente responsivo.
- [x] Efeitos visuais temáticos.
- [x] Mensagens interativas.
- [x] Otimização de performance (carregamento rápido).
